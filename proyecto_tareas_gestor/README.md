# 📋 Gestor de Tareas - Proyecto Kanban

Aplicación web desarrollada como parte de la Evaluación N°2.  
Permite gestionar tareas mediante un tablero tipo **Kanban**.

---

## 🚀 Funcionalidades

- ✅ Agregar tareas mediante formulario
- 🔄 Cambiar estado de tareas (Pendientes → En Progreso → Completadas)
- ❌ Eliminar tareas
- 🔢 Contadores automáticos por columna
- 🎨 Diseño responsive (adaptable a celular y tablet)

---

## 🛠️ Tecnologías utilizadas

- HTML5 (estructura semántica)
- CSS3 (Flexbox, variables, responsive design)
- JavaScript (manipulación del DOM y eventos)
